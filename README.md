🛒 KantoShop – E-commerce Clothing Website

KantoShop is a web-based e-commerce application developed as a university course project.
The website simulates an online clothing store where users can browse products, manage a shopping cart, and place orders.

The project focuses on web development fundamentals, database integration, and building core features of an online shopping platform.

🎯 Overview

KantoShop was developed by a team of 2 members as part of a university course project.

The main goal of the project was to design and develop a basic e-commerce platform that allows users to explore products, manage shopping carts, and simulate the online purchasing process.

The system includes essential features commonly found in modern e-commerce platforms, including:

User authentication

Product browsing

Shopping cart management

Product administration

The project helped improve skills in:

Web development

Database design

Requirement analysis

Team collaboration

UI/UX improvement

🚀 Features
👤 User Authentication

User registration and login system

Basic authentication for accessing user features

🛍 Product Browsing

View a list of clothing products

Display product information such as name, price, and description

🛒 Shopping Cart

📦 Product Management
